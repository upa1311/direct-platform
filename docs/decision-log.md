# Журнал утверждённых решений

## Правила ведения

- Дата исходной фиксации: 2026-07-11.
- В журнал попадают только прямо утверждённые требования.
- Предварительные варианты, предложения и неразрешённые противоречия сюда не переносятся как решения; они находятся в `open-questions.md`.
- Новое решение должно сопровождаться синхронным обновлением затронутых документов.

| ID | Дата | Утверждённое решение |

| DEC-036 | 2026-07-11 | Для онлайн-оплаты `PLATFORM_DRIVER`: `restaurantCommission = foodSubtotalAfterDiscounts × 15%`, а `smallOrderFee = max(0, 100 − restaurantCommission)` в минимальных единицах. |
| DEC-037 | 2026-07-11 | Доплата не определяется неточным правилом «все заказы до $7»; сумма до исчезновения доплаты рассчитывается динамически. |
| DEC-038 | 2026-07-11 | Водитель получает 100% установленной стоимости доставки; Direct не удерживает долю из доставки водителя. |

| ID | Дата | Утверждённое решение |
