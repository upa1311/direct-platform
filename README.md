# Direct

`food-delivery-platform` — локальный кликабельный прототип сервиса доставки еды Direct. Первый город запуска — Бендеры, язык интерфейса — русский.

## Как открыть прототип

В PowerShell внутри папки проекта выполните:

```powershell
npm.cmd run dev
```

Затем откройте в браузере:

```text
http://localhost:3000
```

На главной странице можно выбрать клиентское, ресторанное, водительское или административное рабочее пространство. Все действия работают только с тестовыми данными в `localStorage` браузера и синхронизируются между вкладками.

## Что можно посмотреть

- каталог тестовых ресторанов;
- страницу ресторана и меню;
- добавление блюд в корзину;
- оформление демонстрационного онлайн-заказа;
- экран активного заказа;
- реестр ресторанов в админке;
- редактирование карточки ресторана, меню и доступности;
- публикационные состояния, аудит и предварительный просмотр.

## Ограничения прототипа

- база данных пока не подключена;
- реальные платежи, SMS, звонки и геолокация не выполняются;
- тестовые изменения сохраняются после обновления страницы в текущем браузере;
- спорные бизнес-действия обозначаются как требующие отдельного решения;
- используется тестовая валюта USD и тестовые названия ресторанов/водителей.

## Основные положения продукта

- Один заказ содержит позиции только одного ресторана.
- Поддерживаются `PLATFORM_DRIVER`, `RESTAURANT_DELIVERY` и `PICKUP`.
- Первым полностью реализуется `PLATFORM_DRIVER`.
- Деньги в будущей рабочей системе хранятся целыми минимальными единицами вместе с `currencyCode`.
- Ежедневное управление ресторанами и меню не должно требовать изменения исходного кода.

## Оплата онлайн в текущем прототипе

Основной безналичный способ называется «Оплата онлайн» и хранится как `paymentMethod = ONLINE`. КЛЕВЕР-карта, KleverPay/банковское приложение, платёжная ссылка и QR рассматриваются как возможные каналы после подключения банка.

Реальная банковская интеграция отсутствует. Кнопка подтверждения оплаты является только явно подписанным демонстрационным действием: деньги не списываются. Наличные отключены до отдельного этапа после завершения и проверки водительского маршрута онлайн-заказов.
